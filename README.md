# 最小承接网页 HTML 版

本文件夹保存 Free Pastor Receiving 第一用户阶段的静态单页网页。

## 文件

| 文件 | 用途 |
|------|------|
| `index.html` | 可直接本地打开或部署的单页 HTML |

## 使用方式

1. 双击 `index.html` 本地预览。
2. 将两个 Apply 按钮的链接从 `#apply-placeholder` 替换为真实 Apply 表单 URL。
3. 将页脚 `YOUR_CONTACT_EMAIL` 替换为专用运营邮箱。
4. 发布前按 `../05-人工审批与外部动作记录.md` 做人工审批记录。
5. 发布后把公开 URL 填入 `../09-第一用户承接执行SOP.md` 的链接记录区。

## 部署选项

- Cloudflare Pages
- GitHub Pages
- Netlify
- 任意静态网站托管

## 边界

- 不在本页公开 Before / After / 24h 表单。
- 不在本页公开领受视频链接。
- 不加入 Grace Map、会员、A1-A10/B1、D1/D2、创始人故事或疗效见证。
- 本页只负责解释、边界、Covenant 和 Apply 入口。
